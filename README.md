# DalamudPlugins-TW

台服 FFXIV 專用的 Dalamud 插件倉庫，提供 API12 版本的插件。

## 使用方式

1. 開啟 Dalamud 設定
2. 進入「實驗性功能」
3. 在「自訂插件儲存庫」加入以下 URL：

```
https://raw.githubusercontent.com/magejiaaa/DalamudPlugins-TW/main/repo.json
```

4. 儲存後即可在插件安裝器中看到這些插件

## 可用插件

| 插件 | 版本 | 說明 |
|------|------|------|
| (**Bossmod Reborn**)[https://github.com/FFXIV-CombatReborn/BossmodReborn] | 7.2.5.109 | Mod 載入器與管理工具 |

## 為什麼需要這個倉庫？

台服使用的 Dalamud 版本為 API12，而官方插件倉庫已更新至 API13。這導致許多插件無法在台服上使用。
本倉庫提供這些插件的 API12 相容版本，讓台服玩家也能使用這些實用的插件。
